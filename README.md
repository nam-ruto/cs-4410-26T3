## Getting Started

### Prerequisites
- Python 3.x
- Jupyter Notebook

### Installation
1. Clone this repository:
```bash
git clone <repository-url>
cd CS-4410
```
2. Install required packages for each assignment:
```bash
cd hw1
pip install -r requirements.txt
```
3. Launch Jupyter Notebook:
```bash
jupyter notebook
```
